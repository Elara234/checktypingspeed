Typing Speed Test
A Python based command line application to measure your typing speed and accuracy. Fun and interactive project to practice typing and test your speed.

Features
- Measures typing speed in Words Per Minute (WPM)
- Calculates accuracy of typed text
- Interactive CLI application
- Beginner-friendly Python project
- Uses standard Python libraries (no external dependencies)

 Tech Stack
- Python
- Standard libraries: `time`
